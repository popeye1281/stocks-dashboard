# Portfolio P&L Tracker

FIFO-based Portfolio Profit & Loss tracker for Indian stocks (NSE/BSE) with LTCG/STCG bifurcation.

## Features

- **Realized P&L** - Trade-level P&L with LTCG/STCG classification
- **Current Holdings** - Unrealized gains with live prices via Yahoo Finance
- **Today's P&L** - Intraday P&L comparing previous close vs live price
- **Portfolio Summary** - Allocation, top gainers/losers, statistics

## Deployment on Streamlit Cloud

1. Push this folder to a GitHub repository
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Click "New app" and select your repo
4. Set main file path: `app.py`
5. Deploy

## Local Development

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Updating Data

To update portfolio transactions or prices:
1. Replace `data/portfolio.csv` with your latest portfolio export
2. Replace `data/latest_prices.csv` with latest prices (generated by `stock_data_refresh.py`)
3. Commit and push to GitHub - Streamlit Cloud auto-redeploys
